Logical Buttons

Matrix-channel logical buttons 15 extra buttons on matrix CH7, not on the transmitter graphic

Mix a transmitter logical switch (or any spare source) onto the matrix channel, then set the value it outputs below. When the matrix channel reaches that value (±10), the logical button fires — with the same per-mode, multi-tap actions as the physical buttons. Switch the mode tabs to map different actions per mode; Configure actions opens the standard editor. Leave the trigger blank to disable a slot.

Transmitter

Model which physical transmitter you're using

The model determines which SVG is displayed, which controls appear in calibration, and the default channel assignments for each control. Calibration still overrides per-control channels individually, so a custom TX mixer setup still works — the model just provides sensible starting points.

X20 hardware options

Check this if your X20 has the optional 3-axis gimbal upgrade — adds the J5 (left-stick twist) / J6 (right-stick twist) axes and the two stick-click momentary buttons (matrix slots 19 / 20) to the axes table, assignment list, and calibration walk. Leave unchecked on a standard 2-axis X20.

HCR Destination

All HCR actions route through this destination. Set once, save to board.

MP3 Trigger Destination

All MP3 Trigger actions route here. Serial = MP3 Trigger wired to this controller's S3/S4. WCB = wired to a WCB (configured there with ?MP3,S<port>). Set once, save to board.

Maestro Locations

Defines where each logical Maestro lives. Buttons/knobs only pick "Maestro 1-8" and don't care about the wiring. Every physical Maestro must have a unique Pololu device # (0-127) set in Maestro Control Center — the value here must match it so only the addressed Maestro responds.

Aux Serial Baud

Maestro (S2) = local Pololu Maestro bus (up to 115200). S3/S4 = aux ports for HCR-local / MP3-local / Serial actions (one port = one device = one baud, ≤57600). Reboot (or re-save) for changes to take effect.

General

Selects the GPIO pin map for your hardware. Save, then reboot the board for a change to take effect — pins are assigned at boot. Default: NaviCore v2.
