HCR Destination

All HCR actions route through this destination. Set once, save to board.

MP3 Trigger Destination

All MP3 Trigger actions route here. Serial = MP3 Trigger wired to this controller's S3/S4. WCB = wired to a WCB (configured there with ?MP3,S<port>). Set once, save to board.

Maestro Locations

Defines where each logical Maestro lives. Buttons/knobs only pick "Maestro 1-8" and don't care about the wiring. Every physical Maestro must have a unique Pololu device # (0-127) set in Maestro Control Center — the value here must match it so only the addressed Maestro responds.

Aux Serial Baud

Maestro (S2) = local Pololu Maestro bus (up to 115200). S3/S4 = aux ports for HCR-local / MP3-local / Serial actions (one port = one device = one baud, ≤57600). Reboot (or re-save) for changes to take effect.
